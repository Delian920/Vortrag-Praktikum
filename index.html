CSS

Es gibt nicht nur HTML sondern auch CSS (Cascading Style Sheets),
Es ist nichts anderes als das Designen von einer Webseite.
es wird dazu benutzt um Farben Schriftarten, ect. zu verändern um die Webseite
besser aussehen zu lassen.
Z.B bedeutet der Code (background-color: #f1f1f1f1;) dass die Hintergrundfarbe auf ein helles Grau umgestellt wird
